JavaScript Crash Course

1. Data Types & Variables

Data Types help us organize data in JavaScript. Here are some examples:

A. String

Strings are sequences of characters used to represent text.

  • "Hello World"
  • `Frontend Simplified is amazing`
  • '100+'
// Coding Example:
let str = 'Casey';
console.log(str[5 - 1]); // Outputs: 'y'

B. Number

Numbers represent numeric values.

  • Arithmetic operations: * (multiplication), / (division)
  • Example: 7 * (4 + 9) = 91
  • Modulus operator: % (e.g., 15 % 2 to determine if a number is even or odd)

C. Boolean

Booleans represent logical entities and can have two values: true or false.

D. Undefined

A variable that has been declared but not assigned a value is undefined.

E. Null

null is an assignment value that represents no value.

F. NaN

NaN stands for "Not-a-Number" and indicates an invalid number.

Note: Getting NaN typically indicates a problem in your code where you're performing a numeric operation on non-numeric data.

G. Variables

Variables store data values. There are three ways to declare variables:

  • var - Outdated and not recommended.
  • let - Can be reassigned; block-scoped.
  • const - Constant; cannot be reassigned; block-scoped. 
// Variable Examples:
let fullName = 'Casey Sexton';

let celsius = 10;
let fahrenheit = celsius * 1.8 + 32;
console.log(fahrenheit); // Outputs: 50

H. Equality

Comparison operators are used to compare values:

  • == - Equal to (compares values).
  • === - Strict equal to (compares values and types).
  • != - Not equal to.
  • !== - Strict not equal to.
  • > - Greater than.
  • < - Less than.
  • >= - Greater than or equal to.
  • <= - Less than or equal to.
// Equality Code Example:
let subscribed = false;
let loggedIn = true;

if (subscribed === true) {
    console.log("Show the video");
} else if (loggedIn === true) {
    console.log("Tell user to upgrade their subscription");
} else {
    console.log("Tell user to subscribe");
}

2. Loops

Loops are used to execute a block of code repeatedly.

A. For Loop

The for loop has the following syntax:

for (initialization; condition; increment) {
    // code block to be executed
}

Example:

for (let i = 0; i <= 99; i++) {
    console.log(i);
}

B. While Loop

The while loop repeats a block of code as long as a specified condition is true.

C. Do While Loop

The do...while loop is similar to the while loop, but it executes the code block at least once before checking the condition.

D. Looping Through a String

Print out each character in a string:

const str = "Frontend Simplified";

for (let i = 0; i < str.length; i++) {
    console.log(str[i]);
}

E. The Big Interview Question

FizzBuzz implementation:

for (let i = 1; i <= 20; i++) {
    if (i % 3 === 0 && i % 5 === 0) {
        console.log(`${i} -> Frontend Simplified`);
    } else if (i % 3 === 0) {
        console.log(`${i} -> Frontend`);
    } else if (i % 5 === 0) {
        console.log(`${i} -> Simplified`);
    } else {
        console.log(`${i} -> ${i}`);
    }
}

3. Functions

A function is a block of code designed to perform a particular task.

A. Function Definition and Invocation

// Function definition
function welcomePersonToFES(firstName, lastName) {
    console.log(`Welcome to Frontend Simplified, ${firstName} ${lastName}`);
}

// Function invocation
welcomePersonToFES('David', 'Brag');
welcomePersonToFES('Zen', 'Doe');
welcomePersonToFES('Paul', 'Bunuon');

Note: Arguments must match the parameters in both order and quantity.

B. Returning Values from Functions

Example of using a value returned by a function:

function convertCelsiusToFahrenheit(celsius) {
    return celsius * 1.8 + 32;
}

console.log(convertCelsiusToFahrenheit(0));  // Outputs: 32
console.log(convertCelsiusToFahrenheit(10)); // Outputs: 50
console.log(convertCelsiusToFahrenheit(30)); // Outputs: 86

4. Arrays

Arrays are data structures that can hold multiple values in one variable.

A. Array Initialization

let arr = [20, 30, 40, 50, 100];
console.log(arr[0]);                // Outputs: 20
console.log(arr[arr.length - 1]);   // Outputs: 100

arr.push(200);
console.log(arr);                   // Outputs: [20, 30, 40, 50, 100, 200]

B. Filtering Arrays

// Using filter method
let newArr = arr.filter(element => element < 50);
console.log(newArr); // Outputs: [20, 30, 40]

C. Mapping Arrays

// Using map method
let newArray = arr.map(element => element * 100);
console.log(newArray); // Outputs: [2000, 3000, 4000, 5000, 10000, 20000]

5. Objects

Objects are used to store multiple properties in one variable.

A. Object Initialization

let users = [
  {
    username: 'Casey',
    email: 'c.blaine.sexton@gmail.com',
    password: 'Bizzle',
    subscription: 'VIP',
    lessonCompleted: [0, 1],
  },
  {
    username: 'Dude',
    email: 'c.blaine.s@gmail.com',
    password: 'Buggy',
    subscription: 'VIP',
    lessonCompleted: [0, 1],
  },
];

B. Accessing Object Properties

// Function to log in a user
function login(email, password) {
  for (let i = 0; i < users.length; i++) {
    if (users[i].email === email) {
      if (users[i].password === password) {
        console.log('Log the user in - the details are correct');
      } else {
        console.log('Incorrect password');
      }
      return;
    }
  }
  console.log('The user could not be found');
}

login('c.blaine.sexton@gmail.com', 'Bizzle');

C. Adding Objects to an Array

// Function to register a new user
function register(user) {
  users.push(user);
}

register({
  username: 'Casey',
  email: 'c.blaine@gmail.com',
  password: 'Bizzy',
  subscription: 'VIP',
  discordId: 'C@discord.com',
  lessonCompleted: [0, 1, 2],
});

console.log(users);

6. DOM (Document Object Model)

The DOM allows you to access and change the styling and content of elements on your website.

A. Accessing DOM Elements

First way (most used):

document.querySelector('#title'); // Using ID
document.querySelector('.title'); // Using class
document.querySelector('h1');     // Using tag

Second way:

document.getElementById('title');

B. Modifying DOM Elements

// Changing inner HTML
document.querySelector('#title').innerHTML = 'Frontend Simplified';

// Appending to inner HTML
document.querySelector('#title').innerHTML += ' is amazing!';

C. Changing CSS Styles

document.querySelector('.title').style.color = 'blue';

D. Example: Changing Elements Using JavaScript

HTML:

<button onclick="changeTitleToRed()">Click Me</button>
<button onclick="turnToBlack()">Turn To Black</button>

CSS:

* {
    transition: 300ms ease;
}

.dark-theme {
    background-color: black;
}

.dark-theme * {
    color: white;
}

.turn-red {
    color: red;
}

JavaScript:

function changeTitleToRed() {
  document.querySelector('.title').classList.toggle('turn-red');
}

function turnToBlack() {
  document.querySelector('body').classList.toggle('dark-theme');
}

Note: Ensure your JavaScript file is loaded after the HTML content by adding defer to the script tag: 

<script src="app.js" defer></script>
